Jogo simples em javascript

Aqui temos um jogo simples criado com javascript. Você pode mover o player e tem um NPC que se move automaticamente

A arena, obstáculos, o player, o NPC e seus movimentos sao criados via javascript

Para mexer o player (0.0) use as setas do teclado

O NPC(-_-) se mexe aleatoriamente sempre que possível usando setInterval (800ms)

Se ocorrer algum erro no posicionamento dos personagens, eles serão tratados via try-catch do javascript
