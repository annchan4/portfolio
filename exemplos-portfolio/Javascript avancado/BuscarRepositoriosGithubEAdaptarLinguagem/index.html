Busca por repositórios github

Busque por repositórios do github e mude a língua deste website usando o menu perto da busca

Será feita uma busca via fetch() na api do github

A língua selecionada será armazenada para futuros acessos usando o localStorage! Tente mudá-la, fechar o navegador e abrir novamente

Os resultados da pesquisa serão armazenado para futuros acessos usando o sessionStorage! Tente fazer uma busca e dar f5(se fechar o navegador, dados irão se perder)

Ao olhar o código fonte do projeto, perceberá que é demonstrado meu conhecimento em módulos, onde os scripts css se acessam
