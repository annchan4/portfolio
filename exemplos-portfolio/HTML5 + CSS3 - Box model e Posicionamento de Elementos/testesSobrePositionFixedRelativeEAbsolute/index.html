Usando o position:relative para alterar a posição de uma caixa dentro da outra...

A)Antes

B)Com relative, top:30px; e left:20px;

Independente de ter pai ou não, esse quadrado de position absolute vai se mexer de acordo com a posição da página como um todo, ignorando quaisquer outros elementos. Mas diferentemente so fixed, ele não vai permanecer no lugar quando eu fizer scroll down

Quadrado com position relative, left:-20px; mas z-index menor que o quadrado que o envolve (quadrado azul vai ficar por trás)
